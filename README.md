# GRAPHQL
Chatie GraphQL Schema &amp; Settings

Graph Backend as a Service - GraphCool: https://graph.cool

## Setup Environment

```shell
sudo npm install -g graphcool
```

## Development

```shell
graphcool playground
```

## Deploy

```shell
graphcool deploy
```
